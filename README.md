# GhanaPulse - Ghana News Portal

![GhanaPulse Logo](images/logo.png)

A modern news website covering Politics, Sports, and Entertainment in Ghana.

## Features
- Netlify CMS integration for content management
- Responsive design (mobile-friendly)
- Ghana-themed color scheme (red, gold, green)
- Category-based news sections

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/ghanapulse.git
   ```
2. Install dependencies (if any):
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm start
   ```

## Deployment
This site is automatically deployed via Netlify when pushing to the `main` branch.

[![Netlify Status](https://api.netlify.com/api/v1/badges/YOUR-DEPLOY-ID/deploy-status)](https://app.netlify.com/sites/ghanapulse/deploys)

## Content Management
Access the CMS at:  
`https://ghanapulse.netlify.app/admin/`
